# สถานีสุขภาพชุมชนตำบลคำใหญ่

เว็บแอพคัดกรอง NCDs สำหรับประชาชนและ อสม. ใช้งานบนมือถือได้ดี มีทะเบียนประชากรอายุ 35 ปีขึ้นไปจากไฟล์ CSV ที่แนบมา และบันทึกผลลง `localStorage` ก่อนส่งต่อ Google Sheet ผ่าน Google Apps Script

## เปิดใช้งานในเครื่อง

```powershell
node server.js
```

จากนั้นเปิด `http://localhost:4173`

## ระบบผู้ใช้งาน

- ผู้ดูแลเริ่มต้น: `admin`
- รหัสผ่านเริ่มต้น: `123456`
- หลังเข้าสู่ระบบครั้งแรก ระบบจะบังคับเปลี่ยนรหัสผ่าน
- บัญชีที่สมัครใหม่จะเป็นสถานะรออนุมัติ ต้องให้ผู้ดูแลเปิดเมนู `ผู้ใช้งาน` แล้วกดอนุมัติ
- ผู้ดูแลสามารถสร้างบัญชี อสม. จากรายชื่อ `อสม.ที่รับผิดชอบ` ในทะเบียนประชากรได้ โดยระบบจะสร้าง username แบบ `v001`, `v002` และตั้งรหัสผ่านเริ่มต้น `123456`
- หากมีไฟล์บัญชีจากระบบข้อมูลสุขภาพชุมชน ให้นำเข้าในเมนู `ผู้ใช้งาน` โดยใช้คอลัมน์ `username,password,fullName,role,status,volunteerName`
- มีไฟล์ตัวอย่างสำหรับนำเข้าอยู่ที่ `data/community-users-template.csv`

> หมายเหตุ: เวอร์ชัน static นี้เก็บบัญชีและผลคัดกรองใน browser localStorage จึงเหมาะกับต้นแบบ/ใช้งานภายในแบบควบคุมเครื่อง หากใช้งานจริงหลายเครื่องควรย้ายระบบ login และฐานข้อมูลไปไว้ฝั่ง server หรือ Apps Script พร้อมมาตรการคุ้มครองข้อมูลส่วนบุคคล

## เชื่อม Google Sheet

1. เปิด Google Apps Script แล้วสร้างโปรเจกต์ใหม่
2. วางโค้ดจาก `google-apps-script/Code.gs`
3. Deploy เป็น Web app
4. ตั้งค่า Execute as: `Me`
5. ตั้งค่า Who has access: ตามการใช้งานจริง เช่น `Anyone with the link` สำหรับเครื่องภาคสนาม
6. Copy Web App URL ไปใส่ในเมนู `ตั้งค่า Sheet` ของเว็บแอพ

Sheet ID ที่ตั้งไว้แล้ว:

```text
1yydaayto6uVSTJ8Qav84kBcZ_HDQiBGKBIkBOGjlBvU
```

## หมายเหตุด้าน AI

ส่วน `AI Health Brief` ในเวอร์ชันนี้เป็น rule-based clinical decision support สำหรับคัดกรองเบื้องต้น ทำงานได้ทันทีโดยไม่ต้องส่งข้อมูลสุขภาพไป API ภายนอก ถ้าต้องการใช้ LLM จริง ควรวางไว้ฝั่ง server หรือ Apps Script พร้อมการยินยอมและมาตรการคุ้มครองข้อมูลส่วนบุคคลก่อนใช้งานจริง
